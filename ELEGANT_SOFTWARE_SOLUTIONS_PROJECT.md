## 1. Project Overview

### What Is This Project?

The **Phoenix Platform** (codename: rising from legacy Wix site) is a production-grade web application that serves as the digital home for Elegant Software Solutions - an 18+ year enterprise software company that has pivoted to become an AI enablement services provider headquartered in Alpharetta, Georgia.

### Platform Capabilities

The Phoenix Platform provides:

1. **Service Marketing**: Showcase AI enablement services with clear pricing
2. **Thought Leadership**: 95+ blog articles establishing AI expertise
3. **Lead Generation**: Contact forms, newsletter, Calendly integration
4. **Content Management**: Full admin dashboard for blog publishing
5. **Semantic Search**: AI-powered content discovery using OpenAI embeddings
6. **Regional Presence**: Location-specific pages for Atlanta, Alpharetta, Orlando

### Problem Statement

Before Phoenix, Elegant Software Solutions had:
- An outdated Wix website with limited functionality
- No content marketing or thought leadership presence
- Manual lead capture processes
- No AI-powered features to demonstrate expertise

### Solution

Phoenix addresses these challenges with:
- Modern Next.js 15 architecture with React 19
- Comprehensive blog system with 95 articles
- AI-powered semantic search
- Automated newsletter digest delivery
- Integrated scheduling (Calendly)
- Slack-connected lead notifications
